概要

このページでは、Storybook の button.stories.ts で定義されたボタンコンポーネントのすべてのバリエーションを確認できます。

各ボタンは、サイズ、色、状態の組み合わせで様々なユースケースに対応しています。

プロパティ一覧

button.stories.ts で定義されているプロパティの詳細です。

プロパティ 型 デフォルト値 説明
label string - ボタンに表示されるテキスト（必須）
size "small" | "medium" | "large" "medium" ボタンのサイズ
variant "primary" | "secondary" | "danger" | "success" "primary" ボタンのスタイルバリエーション
disabled boolean false ボタンの無効状態
onClick () => void - クリック時のイベントハンドラ
